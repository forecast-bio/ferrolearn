# ferrolearn-neural

Neural network estimators for the [ferrolearn](https://github.com/dollspace-gay/ferrolearn) machine learning framework.

A scikit-learn equivalent for Rust, providing:

- `MLPClassifier` — multi-layer perceptron classifier with backprop.
- `MLPRegressor` — multi-layer perceptron regressor with backprop.
- `BernoulliRBM` — Bernoulli Restricted Boltzmann Machine trained via CD-1.
